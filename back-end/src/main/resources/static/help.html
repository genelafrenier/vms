Frequently Asked Questions

Here you can find answers to your most common questions.

Where do I find the events I previously volunteered for?

You will find this information under 'Event History' in the User Dashboard.

As an organizer, how do I create an event posting?

On the 'Organizer Dashboard' there is a plus button to create an event.

How do I apply for an event?

There is an apply button inside the event details when you click on the event.

How do I get organizer access to create an event posting?

Contact your corresponding department to verify your account with the career center.

If I have questions about an event, where can I ask them?

On the event details page, there is a section for any questions and comments when you apply for the event.

How do I edit an event posting?

Only organizers can edit events, which can be found on their dashboard under 'My Events.'
